HTML to SCSS

Node.js process that parses HTML to Sass SCSS.

Things to Note: The parser is a work in progress but will accurately represent the DOM. You should use SMACCS for scalability.

Learn more about the project, including the npm package at the GitHub project page.
Sometimes the server dies, and, well, that's just no fun but it would help if you submit an issue on GitHub with the input value.
